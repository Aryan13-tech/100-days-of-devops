# 🚀 Day 07 - Passwordless SSH Setup

## 📌 Challenge Overview

The **xFusionCorp Industries system administrators** configured scripts on the **Jump Host** to automate tasks across all application servers in the **Stratos Datacenter**.

To ensure the automation scripts work properly, the **thor user on the jump host** must have **password-less SSH access** to all application servers using their respective sudo users.

---

# 🎯 Objective

Configure **passwordless SSH authentication** from:

```
thor (Jump Host)
```

to all app servers:

| Server       | Hostname | User   |
| ------------ | -------- | ------ |
| App Server 1 | stapp01  | tony   |
| App Server 2 | stapp02  | steve  |
| App Server 3 | stapp03  | banner |

---

# 🖥 Infrastructure Details

| System       | Hostname  | User   |
| ------------ | --------- | ------ |
| Jump Host    | jump_host | thor   |
| App Server 1 | stapp01   | tony   |
| App Server 2 | stapp02   | steve  |
| App Server 3 | stapp03   | banner |

---

# 🔑 Step 1 — Generate SSH Key on Jump Host

Login to the jump host as **thor** and generate an SSH key.

```bash
ssh-keygen -t rsa
```

Press **Enter** three times to accept default values.

This generates:

```
~/.ssh/id_rsa
~/.ssh/id_rsa.pub
```

---

# 📡 Step 2 — Copy SSH Key to App Server 1

```bash
ssh-copy-id tony@stapp01
```

Enter password when prompted.

---

# 📡 Step 3 — Copy SSH Key to App Server 2

```bash
ssh-copy-id steve@stapp02
```

Enter password when prompted.

---

# 📡 Step 4 — Copy SSH Key to App Server 3

```bash
ssh-copy-id banner@stapp03
```

Enter password when prompted.

---

# 🔍 Step 5 — Verify Passwordless SSH

Test login without entering passwords.

```bash
ssh tony@stapp01
exit

ssh steve@stapp02
exit

ssh banner@stapp03
exit
```

If the login works **without asking for passwords**, the setup is successful.

---

# ⚙ How Passwordless SSH Works

1. SSH key pair is generated.
2. The **public key** is copied to the remote server.
3. The server stores the key in:

```
~/.ssh/authorized_keys
```

4. When connecting, SSH verifies the key and allows login without password.
