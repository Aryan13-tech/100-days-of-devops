# 🚀 Day 11 – Tomcat Deployment (KodeKloud Engineer)

## 📌 Task Summary

* Install **Tomcat** on App Server 2 (`stapp02`)
* Configure Tomcat to run on **port 6300**
* Copy `ROOT.war` from **jump host (/tmp)**
* Deploy application
* Verify using `curl`

---

## 🖥️ Infrastructure Used

| Server    | Purpose                               |
| --------- | ------------------------------------- |
| jump-host | Access server                         |
| stapp02   | Application server (Tomcat runs here) |

---

## ⚙️ Step-by-Step Commands

### 🔹 1. SSH into App Server 2

```bash
ssh steve@stapp02
```

---

### 🔹 2. Install Tomcat

```bash
sudo yum install -y tomcat
```

---

### 🔹 3. Change Port to 6300

```bash
sudo vi /etc/tomcat/server.xml
```

Find:

```xml
Connector port="8080"
```

Replace with:

```xml
Connector port="6300"
```

Save & exit:

```
Esc → :wq
```

---

### 🔹 4. Start & Enable Tomcat

```bash
sudo systemctl start tomcat
sudo systemctl enable tomcat
```

---

### 🔹 5. Copy WAR File (from Jump Host)

```bash
scp /tmp/ROOT.war steve@stapp02:/tmp
```

---

### 🔹 6. Move WAR to Webapps

```bash
sudo mv /tmp/ROOT.war /usr/share/tomcat/webapps/
```

---

### 🔹 7. Restart Tomcat

```bash
sudo systemctl restart tomcat
```

---

### 🔹 8. Verify Deployment

```bash
curl http://stapp02:6300
```

✅ Expected Output:

```html
<h2>Welcome to xFusionCorp Industries!</h2>
```

---

## ⚠️ Common Mistakes

❌ Running `systemctl` on jump host
✔️ Always run on `stapp02`

❌ Wrong password during `scp`
✔️ Use correct credentials

❌ Not changing port properly
✔️ Verify in `server.xml`
